元素发现历史

从古代到现代，人类对元素的探索历程

古代 (公元前)

早期元素的发现

古代人类发现并使用了一些简单的元素，主要用于工具制造和装饰品。

金 (Au) 银 (Ag) 铜 (Cu) 铁 (Fe) 锡 (Sn) 铅 (Pb) 硫 (S) 碳 (C)

这些元素大多以单质形式存在于自然界中，或通过简单的冶炼工艺获得。

17世纪

近代化学的开端

随着科学实验方法的建立，更多元素被系统地发现和研究。

磷 (P) 砷 (As)

1669年，德国炼金术士亨尼希·布兰德通过蒸馏尿液发现了磷元素。

18世纪

气体元素的发现

科学家们开始系统地研究气体，发现了多种气体元素。

氢 (H) 氧 (O) 氮 (N) 氯 (Cl) 氟 (F)

1766年，亨利·卡文迪许发现氢气；1774年，约瑟夫·普里斯特利发现氧气。

19世纪初

碱金属和碱土金属的发现

通过电解法，科学家们发现了活泼的金属元素。

钠 (Na) 钾 (K) 钙 (Ca) 镁 (Mg)

1807年，汉弗莱·戴维通过电解熔融的碱金属和碱土金属的氢氧化物，成功分离出钠和钾。

19世纪中期

元素周期表的诞生

门捷列夫根据元素的原子量和化学性质，编制了第一张元素周期表。

1869年，德米特里·门捷列夫发表了元素周期表，预测了当时尚未发现的元素。

19世纪末

稀有气体的发现

科学家们发现了性质稳定的稀有气体元素。

氦 (He) 氖 (Ne) 氩 (Ar) 氪 (Kr) 氙 (Xe)

1894年，瑞利勋爵和威廉·拉姆齐发现了氩气，随后又发现了其他稀有气体。

20世纪初

放射性元素的发现

居里夫妇等科学家发现了放射性元素，开启了核化学的研究。

钋 (Po) 镭 (Ra) 锕 (Ac)

1898年，玛丽·居里和皮埃尔·居里发现了钋和镭元素。

20世纪中期

人工合成元素的开始

通过核反应，科学家们开始合成自然界中不存在的元素。

锝 (Tc) 钷 (Pm) 镎 (Np) 钚 (Pu)

1937年，卡洛·佩里埃和埃米利奥·塞格雷首次合成了锝元素，这是第一个人工合成的元素。

20世纪后期

超铀元素的合成

科学家们合成了更多的超重元素，扩展了元素周期表。

镅 (Am) 锔 (Cm) 锫 (Bk) 锎 (Cf)

这些元素大多是在粒子加速器中通过重离子碰撞合成的。

21世纪

超重元素的探索

科学家们继续探索周期表的边界，合成更重的元素。

鿏 (Mt) 𫟼 (Ds) 𬬭 (Rg) 鎶 (Cn) 鿬 (Ts) 鿏 (Og)

这些超重元素的半衰期极短，需要复杂的实验设备和技术来合成和检测。
