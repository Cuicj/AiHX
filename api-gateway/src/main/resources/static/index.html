第一部分：初级化学认知——推开物质世界的大门

基础层 · 建立化学思维的基础框架

物质"变身"科学

  • • 化学变化生成新物质，区别于物理变化
  • • 核心三问：物质构成？如何变化？伴随什么？
  • • 实用价值：解释"饭煮熟""铁生锈"等日常现象

物质分类

  • • 纯净物（单质、化合物）与混合物
  • • 常见元素：O、C、H、Fe、Ca等生活高频元素
  • • 基本反应：化合、分解、置换反应类型
  • • 变化特征：发光、发热、颜色变化、气体生成

简单实验

  • • 蜡烛燃烧、铁生锈、醋泡鸡蛋壳
  • • 基础工具：天平、量筒、pH试纸的使用
  • • 化学语言：元素符号、化学式、方程式

第二部分：生活中的化学——化学与日常的深度绑定

应用层 · 化学知识在日常生活中的具体应用

衣食住行中的化学

食

  • • 食物消化：淀粉→葡萄糖，蛋白质氧化供能
  • • 食品添加剂：防腐剂、增味剂、色素
  • • 烹饪化学：加醋去腥、面团发酵

衣

  • • 纤维分类：天然纤维vs合成纤维
  • • 聚合反应：涤纶、尼龙的形成
  • • 洗涤原理：表面活性剂、漂白剂

住

  • • 建筑材料：水泥水化、玻璃熔融
  • • 室内环境：甲醛危害、活性炭吸附
  • • 涂料防腐：化学防护原理

行

  • • 能源化学：燃烧反应与能量转化
  • • 电池原理：充放电化学机制
  • • 金属防护：涂层、电镀技术

健康与医药中的化学

  • • 人体化学平衡：血液pH缓冲、电解质功能
  • • 药物化学：阿司匹林合成、中药成分提取
  • • 消毒防疫：酒精、84消毒液原理

环境与能源中的化学

  • • 环境问题：酸雨、温室效应、白色污染
  • • 能源转型：化石燃料、可再生能源、氢能
  • • 绿色化学：原子经济性、清洁生产
